The file being submitted is tokenizer.rb.
This file contains the main method as well as the tokenizer function.
To run this program, type "ruby tokenizer.rb (filename)" where filename is
the name of the file to be converted to tokens. The results will be printed to 
the console.
This command can be typed in the stdlinux after the file has been copied into the current directory, or in the command
prompt for Windows if ruby is installed on the machine.
